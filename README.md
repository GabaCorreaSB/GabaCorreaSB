Software engineer with a background in network infrastructure and data protection, now deep into cybersecurity. I've been working across fintech and enterprise IT in Brazil before moving to the US, and most of what I actually know came from breaking things in production and figuring out why.

Right now I'm finishing a Master's in Cybersecurity Management at University of Fairfax and working toward CISSP. My projects tend to live at the intersection of IoT, security, and automation, because building something that's fast but insecure isn't really building something useful.

### Projects

**[LoR4Sun](https://github.com/Data-Tech-Team/Lor4sun):** LoRaWAN monitoring system for photovoltaic installations, with real-time web dashboards.

**[Coin Fetcher](https://github.com/GabaCorreaSB/coin-fetcher):** Go microservice that pulls crypto data from CoinGecko. Built to be fast and dumb-simple.

**[FeedWeb](https://github.com/GabaCorreaSB/Feed-web):** Real-time currency tick visualization and analytics platform.
